# Centros de Cuidado DIF Jalisco

Base de datos completa de centros de atención y cuidado en Jalisco, México.

## Descripción

Este repositorio contiene la base de datos de centros de cuidado del Sistema DIF Jalisco, incluyendo:

- **2,188 centros registrados**
- **115 municipios de Jalisco**
- **22 tipos de servicios diferentes**
- **50 campos de información**

## Archivos

- `cc_dif_final.xlsx` - Base de datos completa en formato Excel
- `index.html` - Página web para descarga del archivo
- `image.png` - Logo DIF Jalisco

## Características

- Nombres oficiales verificados mediante Google Maps Places API
- Información geográfica completa (coordenadas y direcciones)
- Clasificación por sector (Público, Privado, Institución de la sociedad civil)
- Filtros preparados para dashboard de ArcGIS:
  - ¿En dónde lo está buscando? (Municipio)
  - ¿Qué está buscando? (Nombre del centro)
  - ¿Para quién lo está buscando? (Tipo de servicio)

## Fuentes de Datos

1. Sistema Nacional para el Desarrollo Integral de la Familia (DIF)
2. Google Maps Places API
3. DIF Jalisco y DIF Municipales
4. INEGI - DENUE (Directorio Estadístico Nacional de Unidades Económicas)

## Uso

Visita la página web en GitHub Pages para descargar la base de datos completa.
